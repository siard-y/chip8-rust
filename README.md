# CHIP-8 emulator

### Requirements
SFML:
- `sudo apt install libsfml-dev libcsfml-doc`
- `sudo pacman -S sfml csfml`
  
 
### Compiling
`cargo build --release`


### Run
`./target/release/chip8-rust ~/<dir to CHIP-8 games>`


[Download CHIP-8 game pack](https://www.zophar.net/pdroms/chip8/chip-8-games-pack.html)
